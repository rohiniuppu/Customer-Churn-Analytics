# Customer Churn & Employee Sentiment Analytics Dashboard

## Project Overview

Designed an interactive Power BI dashboard to analyze employee reviews, customer sentiment, leadership perception, work-life balance, and business outlook using Glassdoor review data. The project demonstrates end-to-end data analytics including data cleaning, transformation, visualization, and business insight generation.

---

## Dataset

- Source: Glassdoor Employee Reviews Dataset
- Sample Size: 1,040 employee reviews (sampled from a larger dataset)
- Tool Used: Power BI

---

## Technologies Used

- Power BI
- Power Query
- SQL
- Microsoft Excel

---

## Data Cleaning

Performed preprocessing using Power Query:

- Removed duplicate records
- Filtered NULL values
- Removed unnecessary columns
- Split combined columns
- Extracted company names
- Created sentiment categories using conditional columns
- Calculated Pros and Cons length

---

## Dashboard Features

### Dashboard 1 – Employee Overview
- KPI Cards
- Rating Trends
- Sentiment Analysis
- Job-wise Review Distribution
- Interactive Filters

### Dashboard 2 – Culture Scorecard
- Work-Life Balance Analysis
- CEO Approval
- Risk Level Analysis
- Company-wise Culture Comparison

### Dashboard 3 – Leadership Insights
- CEO Approval Analysis
- Business Outlook
- Word Cloud
- Leadership Performance

### Dashboard 4 – Sentiment Deep Dive
- Sentiment Gap Analysis
- Scatter Plot
- Trend Analysis
- Company-wise Emotional Score

---

## Key Insights

- Positive employee sentiment slightly outweighs negative sentiment.
- Companies such as Amazon, TCS, and HP show stronger employee satisfaction.
- CEO approval positively correlates with employee sentiment.
- Work-life balance and career growth significantly influence employee satisfaction.
- Dashboard enables interactive business intelligence reporting.
